Original Map: “Climb of Patience”

Overall structure

The map is one continuous vertical climbing world. The player starts at the bottom inside a cauldron-like container and uses a hammer-like tool to climb.

The world should feel handmade, strange, and hostile. Objects are oversized, awkwardly placed, and often visually funny. The player should always be able to see the next obstacle, but the path should require experimentation.

The map has 9 major zones:

1. Rust Pit
2. Drainpipe Yard
3. Furniture Collapse

5. Roof Maze

Zone-by-zone visual direction

1. Rust Pit

This is the tutorial-like start. The player is enclosed by low walls and scrap metal. The space should feel heavy and dirty. Use dark browns, rust orange, black oil stains, and grey concrete.

Objects should look discarded rather than intentionally placed. Pipes should have dents. Metal sheets should overlap. The first few ledges should be forgiving.

The main skills taught here are:

{
  "skills": [
    "push against floor",
    "hook over pipe",
    "pull body upward",
    "balance on slanted metal"
  ]
}

---

2. Drainpipe Yard

This area should feel like an abandoned construction drainage site. Large circular concrete pipes are stacked unevenly. The player must hook the hammer inside rings and around curved edges.

The main danger is sliding off rounded surfaces. The map should not yet punish the player too hard, but mistakes should drop them to the Rust Pit.

{
  "difficulty": "early-medium",
  "mainChallenge": "curved surfaces and unstable resting points",
  "recommendedColors": ["wet grey", "mud brown", "dark green moss"]
}

---

3. Furniture Collapse

This zone should feel funny and strange. The player climbs household furniture that has somehow formed a vertical tower.

Use sofas, chairs, lamps, shelves, drawers, cushions, and broken table legs. The player should sometimes rest on a comfortable-looking object, then immediately face a difficult narrow hook.

{
  "difficulty": "medium",
  "mainChallenge": "awkward angles and uneven ledges",
  "visualTheme": "domestic objects stacked like a landslide"
}

---

4. Tool Shed Wall

This is a more technical area. Add tools as decorations: saws, hammers, paint cans, clamps, drills, extension cords, and wooden boards.

The key gameplay is precision. Ledges are thinner. Hooks are smaller. The player should need to place the hammer carefully instead of relying on big swings.

{
  "difficulty": "medium-hard",
  "mainChallenge": "small hooks and narrow shelves",
  "failurePenalty": "fall back to Furniture Collapse"
}

---

5. Roof Maze

This is the first major psychological test. The player reaches rooftops and feels high above the starting area. The surfaces are sloped, and there are big side drops.

Rooftops should have gutters, chimneys, antennas, tiles, and satellite dishes. Some slopes should be safe if handled slowly, but launching too hard should cause a large fall.

{
  "difficulty": "hard",
  "mainChallenge": "sloped roofs and dangerous side falls",
  "cameraFeeling": "open and exposed"
}

---

6. Cliffside Junk Trail

The map transitions into a rocky natural cliff with man-made junk embedded in it. The objects should feel like wreckage caught on the mountain.

Use tires, old car parts, warning signs, bent poles, boulders, ropes, and cracked stone platforms.

{
  "difficulty": "hard",
  "mainChallenge": "mixed organic and artificial shapes",
  "failurePenalty": "medium to severe"
}

---

7. Frozen Observatory

This is a cold, slippery late-game section. The player climbs snow, ice, and observatory equipment.

The surfaces should be visually smoother and cleaner than earlier zones. Slippery materials should be used carefully. The player should still be able to climb, but resting should feel insecure.

{
  "difficulty": "very hard",
  "mainChallenge": "low-friction surfaces",
  "visualTheme": "snow, ice, glass, telescope parts"
}

---

8. Antenna Spire

This is the final mechanical challenge. It should be tall, thin, and frightening. The player climbs radio tower crossbars and antenna parts.

Everything is narrow. There are few safe resting places. The player should use precise hammer placement and controlled swings.

{
  "difficulty": "final",
  "mainChallenge": "precision climbing on thin metal objects",
  "failurePenalty": "severe"
}

---

9. The Quiet Beyond

The ending should contrast with the rest of the map. It should be calm, open, and almost empty.

Use pale colors, a flat platform, a small flag, and a quiet sky. This gives the player emotional relief after the climb.

{
  "difficulty": "none",
  "purpose": "ending and emotional release",
  "visualTheme": "empty sky, pale stone, silence"
}
